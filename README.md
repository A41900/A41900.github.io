# Creative Landing Page

Live demo: [portfolio](https://a41900.github.io)

Personal landing page built with JavaScript and Canvas.
The focus of this project is visual clarity and interaction, using a deliberately
simplified motion setup designed for a calm and readable experience.

The underlying motion system was developed separately as an exploratory study
and is documented here:
[Generative Motion Studies](https://github.com/A41900/generative-motion-studies)
